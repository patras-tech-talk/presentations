# Patras Tech Talk 2025.10 [Software, better]

**Date:** Wednesday, October 22, 2025, 18:00 - 21:00
**Venue:** coLab Makerspace, Patras

## Talks

### Do it once. Do it right. Do it again: Data Science Done Right

A talk examining reproducible workflows and best practices in data science.

### The Scarcity of Engineering Craftsmanship

A presentation exploring the importance of true engineering principles and quality in software development.

## About

Focused on building improved technology through discussions on reproducible workflows and engineering craftsmanship. The meetup explored what it means to do software development right.
