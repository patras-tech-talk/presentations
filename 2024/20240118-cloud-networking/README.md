# January event [Cloud networking]

**Date:** Thursday, January 18, 2024, 18:15 - 20:30 EET
**Venue:** Deloitte Patras (Brainzone), 28is Oktovriou Str, Patras

## Talks

### Cruising through Amazon VPC
**Speaker:** Pavlos Kaimakis (AWS)
**Time:** 18:45 - 19:15

A journey through Amazon VPC, providing an introductory presentation on the core AWS networking elements. Learn how to set up your network using Internet Gateway, Security Groups, NACLs, Subnets, and Route tables to host your application in a highly available and secure way.

### Cruising out of the boundaries of your VPC
**Speaker:** Alexandros Soumplis (AWS)
**Time:** 19:30 - 20:00

A cruise beyond the boundaries of VPCs, introducing Transit Gateway and Hybrid DNS, enabling network expansion and taking advantage of hybrid DNS to simplify operations and maximize flexibility.
