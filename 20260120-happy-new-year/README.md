# Patras Tech Talk 2026.01 [Happy New Year]

**Date:** Tuesday, January 20, 2026, 18:00 - 21:00 EET
**Venue:** PwC Technology and Innovation Hub Patras, 28 Oktovriou & Othonos-Amalias, Patras

## Talks

### NixtCloud
**Speaker:** Vaios Karastathis
**Time:** 18:30 - 19:10

Demonstrates how NixOS, Nextcloud, and Holesail combine to transform a USB drive into accessible personal cloud storage.

### Diagram as Code and ADRs: Documenting the 'What' and 'Why'
**Speaker:** Ioannis Skitsas
**Time:** 19:15 - 19:50

Explores treating architecture diagrams like version-controlled code instead of static images, and using Architectural Decision Records to capture design rationale.

## About

The gathering welcomes attendees for icebreaker presentations, discussion, and interaction with the tech community of Patras. The program includes networking, with doors opening at 18:00 and closing at 21:00.

**Hosts:** Artemios Vogiatzis, Marios Karagiannopoulos, Victoria Daskalou, Giannis Skitsas
