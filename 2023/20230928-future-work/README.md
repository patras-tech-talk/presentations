**Speaker:** Artemios Vogiatzis
**Time:** 18:15 - 18:40

An exploration of different career trajectories available in the software industry.

### Open source contribution as mentor and developer: The Google Summer of Code (GSOC) experience
**Speaker:** Ioannis Skitsas
**Time:** 18:40 - 19:15

Sharing experiences from participating in Google Summer of Code, both as a mentor and developer, and the value of open source contribution.
