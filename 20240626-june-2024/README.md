# Patras Tech Talk 2024.06

**Date:** Wednesday, June 26, 2024, 18:00 - 21:00
**Venue:** PwC Technology & Innovation Hub, Patras

## Talks

### Data Imbalances in Machine Learning

A presentation on handling data imbalances in machine learning projects, covering techniques and strategies for dealing with skewed datasets.

### Software Delivery Using Containerization

A talk on modern software delivery practices using containerization technologies like Docker and Kubernetes.

## About

Discussions covering both the data science challenge of imbalanced datasets and modern DevOps practices for software delivery.
